怎么设置sublime 主题

#怎么设置sublime主题
如何安装插件详见：https://packagecontrol.io/installation

  1. 使用软件自带的主题风格
    在> preference->Color Theme (> 首选项->配色方案) 中可以任意选择，选择完即可查看主题风格，不用重启。并且会在> Preferebces.sublime-seetings 中自动保存设定。
  2. 自定义主题Primer
    ###Theme - Primer
    ####怎么安装
    最简单的安装方法是使用 Sublime Package Control,Theme - Primer是Sublime Package Control的列表中的一项.
  3. 打开命令面板,点击> Tools -> Command Palette… (on Mac)
  4. 选择> Package Control:Install Package
  5. 找到 Theme-Primer 点击进入
